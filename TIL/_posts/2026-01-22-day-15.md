## Process of data processing
### (1) 데이터 이해 단계
- 데이터 컬럼 확인 (컬럼의 의미)
- 데이터 유형 (수치형, 범주형)
- 수치형 (정수, 소수)
- 범주형 (unique한 개수와 범주 유형 파악)
- 데이터 샘플 확인

### (2) 데이터 전처리
- 이상치
- 데이터 정제 
- 데이터 타입 변환
- 데이터 형식을 일치 

### (3) EDA (탐색적 데이터 분석)
3-1) 단변량 -> 변수 하나
- 분포 확인
- 수치형 - 히스토그램, 박스플롯 (바이올린 플롯)
- 범주형 - 막대, 도넛그래프   
--> 시각화 + 인사이트

3-2) 다변량 -> 변수간의 관계 확인
- scatter (산점도)를 통한 상관관계
- GroupBy pandas함수들 이용
- 추세 (시간 관련)   
---> 꺽은 그래프   
---> 상관관계 히트맵    
--> 시각화 + 인사이트   
--> 범주형 + 다른 연속형 변수   
--> 막대그래프  

#### (4) 최종 인사이트 도출 및 실행 전략 제안
