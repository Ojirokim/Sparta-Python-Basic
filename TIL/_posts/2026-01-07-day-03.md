## What CAST(value AS UNSIGNED) does in MySQL
CAST(value AS UNSIGNED) tells MySQL to treat a value as a non-negative integer.
- When casting, MySQL:
Parses the value as a number
Removes leading zeros
Stops reading at the first non-numeric character
Returns an integer ‚â• 0
Example:
```mysql
SELECT CAST('01' AS UNSIGNED);   -- 1
SELECT CAST('002' AS UNSIGNED);  -- 2
SELECT CAST('14' AS UNSIGNED);   -- 14
```
Useful when converting numeric strings like '01', '02', '14' into integers.

‚ö†Ô∏è Note: Negative values wrap around to a large number because UNSIGNED cannot store negatives.
```mysql
CAST(col AS SIGNED)   -- allows negatives
CAST(col AS DECIMAL)  -- preserves decimals
```


## Python conditional expression (A if x else B)
Python supports a conditional expression (often called a ternary operator) with the form:
```python
A if x else B
```

How it works:
```python
## If x is truthy, the expression evaluates to A
## If x is falsy, the expression evaluates to B
```
The condition x is not required to be True or False.
Python evaluates x using truthiness rules.

Common falsy values include:
- False
- 0, 0.0
- "" (empty string)
- [], {} (empty collections)
- None

All other values are considered truthy (including negative numbers).
This expression is equivalent to:
```python
if x:
    result = A
else:
    result = B
```
Use conditional expressions for simple, readable decisions, but prefer a full if statement when the logic becomes complex.
