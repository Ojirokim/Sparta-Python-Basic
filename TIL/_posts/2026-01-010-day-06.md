## Python Iterable vs Non-Iterable
An iterable is an object you can loop over using for

### Common iterable examples (MEMORIZE)
```python
# Sequence types
list        # [1, 2, 3]
tuple       # (1, 2, 3)
str         # "abc"
range       # range(5)
```

```python
# Set & mapping types
set         # {1, 2, 3}
frozenset  # frozenset({1, 2})
dict        # iterates over keys
dict.keys()
dict.values()
dict.items()
```

```python
# Other iterables
bytes       # b"abc"
bytearray   # bytearray(b"abc")
enumerate(...)
zip(...)
map(...)
filter(...)
generator   # (x*x for x in range(5))
```


### Common non-iterable examples (MEMORIZE)
```python
# Non-sequence types
int         # 10
float       # 3.14
bool        # True
None        # None
complex     # 2+3j
```

### Exam trap: strings ARE iterable
```python
for c in "hi":
    print(c)
```


## Python Mutable vs Immutable
Mutable = can be changed in place without creating a new object

### Mutable objects (VERY IMPORTANT)
```python
list        # [1, 2, 3]
dict        # {"a": 1}
set         # {1, 2}
bytearray
```

### Immutable objects (VERY IMPORTANT)
```python
int
float
bool
str
tuple
frozenset
bytes
None
```

### Exam trap: immutable container with mutable contents
```python
t = (1, [2, 3])
t[1].append(4)    # allowed
```
